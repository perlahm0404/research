### Priority 12: Create Automated Quality Audit Script ⏱️ 4-6 hours

**Script:** `/Users/tmac/research/scripts/audit-quality.sh`

**Checks:**
1. Line count minimum (400 for TIER-1, 600 for TIER-2/3)
2. Evidence citation count minimum (50 for TIER-1, 75 for TIER-2/3)
3. Required sections present (all 15 from v3.0 prompt)
4. Frontmatter YAML completeness

```bash
#!/bin/bash
# State Research Quality Audit Script

echo "=== Document Line Count Audit ==="
for file in *-MD-*.md *-DO-*.md; do
  lines=$(wc -l < "$file")
  if [ $lines -lt 400 ]; then
    echo "⚠️  $file: Only $lines lines (minimum 400)"
  fi
done

echo ""
echo "=== Evidence Citation Audit ==="
for file in *-MD-*.md *-DO-*.md; do
  citations=$(grep -c "\[FACT\|INFERENCE\|GAP" "$file")
  if [ $citations -lt 50 ]; then
    echo "⚠️  $file: Only $citations citations (minimum 50)"
  fi
done

echo ""
echo "=== Required Section Audit ==="
for section in "Audit & Verification" "CME Tracking" "Lapsed License" "Exemptions"; do
  for file in *-MD-*.md *-DO-*.md; do
    if ! grep -q "## .*$section" "$file"; then
      echo "❌ $file: Missing section '$section'"
    fi
  done
done
```

**Deliverable:** Automated audit script, run monthly before marking states complete
